Part 2: Use HTML Tags to create simple HTML document that a browser can display as a webpage

  • Basics of the Web
  • The web is computers talking to each other
  • HTML is Hypertext Markup Language - it is the text used to create documents and what you see; it was invented in the 1990's; it is the most common document used on the Web in addition to Plain Text, images, videos and music
  • URL’s – Refers to Documents
  • HTTP - the main protocol of the web
  • Web Applications - Grand Finale
  • WWW is the World Wide Web. It is a collection of HTML documents
  • Hypertext markup is how it looks on the Web.
  • References to other documents – e.g. Images and videos, Links to other pages
  • Servers are computers that host the files

Major Pieces

  • You + Your Computer + Browser (Chrome, Firefox, etc.)
  • Send document to The Internet using HTTP
  • Gets it to Servers where it is filed then
  • Sent back to The Internet to Browser to Your Computer where You can read it

HTML Editor

  • Plain text is plain text
  • HTML Mark Ups – made up of Tags (Elements)

Tags

  • Tags - <tag> contents </tag>; <tag> is the opening Tag; </tag> is the closing Tag
  • b Tag is for bold - <b> contents </b>
  • If you only have an opening b Tag (without a closing b Tag) everything after the opening b Tag will be bold
  • em Tag is for emphasis; italics - <em> contents </em>
  • If you only have an opening EM Tag (without a closing EM Tag) everything after the opening EM Tag will be italicized

Attributes

  • <Tag ATTR = “value”> contents </Tag>
  • Anchor Tag <a> - Link
  • < a href = www.reddit.com > derp </a>
  • Href – attribute name
  • www.reddit.com – value
  • Derp – Link
  • Example: This website is my favorite < a href = “Udacity.com” > my favorite </a> - my favorite

Images

  • <img> - images
  • <img src = “url” alt = “text”>
  • img is a Void Tag and has no closing text

White Space

  • Text on multiple lines
  • <br> - break (void)
  • <p> - paragraph - <p> content </p>

Inline / Block

  • <br> - inline: Text <br> text; it only creates 2 lines of text)
  • <p> - block: <p> text </p> text; it creates an invisible box
  • <em> is inline
  • <img> is inline
  • <a> is inline
  • <br> is inline
  • <a> is inline
  • <strong> is inline
  • <form> is block

Span/Div

  • –css class (separate language)
  • <span> - text </span> is inline
  • <div> text </div> is block
  • These elements contain their contents:
  • <Span class =”foo”> text </span>
  • <div class = “bar”> text </div>

HTML Documents

  • <!Doctype HTML> - doctype
  • <html> - tag

Head meta data

  • <head> - head
  • <title> Title! </title> - title
  • </head>

Body

  • <body>
  • <b> content </b>
  • </body>
